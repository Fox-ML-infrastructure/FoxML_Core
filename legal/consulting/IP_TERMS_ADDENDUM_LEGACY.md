This addendum clarifies IP rights for work done under the MCA.

1. Pre-Existing IP

Pre-existing IP owned by Consultant includes:

Fox ML Infrastructure (Fox-v1-infra) system and pipelines

ML research frameworks

HPC infrastructure tooling

Internal development utilities

2. Client-Owned Deliverables

Work created exclusively for the Client under Tier 2 or Tier 3 development becomes client-owned deliverable IP, except where it incorporates Fox ML Infrastructure components.

3. Derivative Works

If deliverables extend or build on Fox ML Infrastructure components:

Client receives an internal-use license to the derivatives.

Core Fox-v1-infra IP remains owned by Consultant.

4. Open-Source Components

Open-source components follow their upstream licenses.

5. Reuse of Non-Proprietary Elements

General engineering patterns, templates, and non-client-specific methods may be reused.

No client data or confidential algorithms will ever be reused.

6. Documentation

Documentation follows the same ownership rules as code.

7. Negotiation

Custom IP terms may be negotiated via SOW.
