# Development Tools

This directory contains development and debugging tools.

## Components

### `provenance.py` - Provenance Tracking
Tracks the provenance (origin and history) of:
- Trading decisions
- Model predictions
- Data sources
- Configuration changes

**Features:**
- Decision tracking
- Model version tracking
- Data lineage
- Audit trail

**Purpose:** Provides full traceability for debugging and analysis.

## Usage

Tools are typically used for:
- Development and debugging
- Performance analysis
- System monitoring
- Audit and compliance

## Integration

Tools integrate with:
- Trading engine for decision tracking
- Model registry for version tracking
- Logging system for audit trails
