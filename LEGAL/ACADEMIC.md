## Common Questions

### Q: Can my PhD student use this for their dissertation research?

**A:** Yes, under AGPL v3 if the project can comply with AGPL copyleft requirements (must publish modifications if deployed as a service). If the research requires proprietary modifications or cannot comply with AGPL, a commercial license is required.

### Q: Can we use this in a class?

**A:** Yes, under AGPL v3 for academic coursework if modifications can be published under AGPL. If the class requires proprietary modifications or closed-source integration, a commercial license is required.

### Q: What about sponsored research?

**A:** If sponsored research can comply with AGPL (publish modifications), AGPL v3 is available. If modifications must remain private or cannot comply with AGPL, a commercial license is required.

### Q: Can students use this for startups?

**A:** If the startup can comply with AGPL (publish modifications), AGPL v3 is available. If the startup requires proprietary modifications or closed-source integration, a commercial license is required.

### Q: What if the university uses it institutionally?

**A:** If institutional use can comply with AGPL (publish modifications), AGPL v3 is available. If institutional use requires proprietary modifications or cannot comply with AGPL, a commercial license is required.
