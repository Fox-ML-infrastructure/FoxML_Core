## Fastest Method (5 minutes)

### Step 1: Convert to PDF

**Option A: Copy to Google Docs**
1. Copy the text from `IP_ASSIGNMENT_AGREEMENT.md`
2. Paste into Google Docs
3. File → Download → PDF

**Option B: Use Markdown to PDF converter**
- Use online tool like markdowntopdf.com
- Or use Pandoc if you have it installed

**Option C: Print to PDF**
- Open the markdown file
- Print → Save as PDF

### Step 2: Sign the PDF (Free Methods)

**Easiest: SmallPDF.com (Free)**
1. Go to https://smallpdf.com/sign-pdf
2. Upload your PDF
3. Click "Add Signature"
4. Choose "Draw" and draw your signature with mouse/trackpad
   - OR choose "Type" and type your name (also valid!)
5. Place signature in both signature fields:
   - First field: "Jennifer Lewis" (as Assignor)
   - Second field: "Jennifer Lewis" (as Assignee for LLC)
6. Fill in dates and title ("Founder" or "Managing Member")
7. Download signed PDF

**Alternative: PDF24.org (Free)**
1. Go to https://tools.pdf24.org/en/sign-pdf
2. Upload PDF
3. Add signature (draw or type)
4. Place in both fields
5. Download

**Mac Users: Preview App (Built-in)**
1. Open PDF in Preview
2. Tools → Annotate → Signature → Create Signature
3. Draw signature with trackpad
4. Click where you want to place it (both signature fields)
5. Save

### Step 3: Fill in Required Fields

Make sure you fill in:
- **Effective Date**: Today's date (e.g., "December 14, 2025")
- **Assignor Date**: Today's date
- **Assignee Date**: Today's date  
- **Title**: "Founder", "Managing Member", or "Owner"

### Step 4: Store Securely

Save the signed PDF:
- In a secure folder on your computer
- In encrypted cloud storage (Google Drive, Dropbox, etc.)
- In your business records folder
- Consider emailing a copy to yourself as backup
