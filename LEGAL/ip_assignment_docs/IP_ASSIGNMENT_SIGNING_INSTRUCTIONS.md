**Quick Guide to Signing the IP Assignment Agreement**

---

## Overview

Since you're the sole owner of Fox ML Infrastructure LLC, you'll be signing in **two capacities**:
1. **As Jennifer Lewis (Individual)** - Assignor (the person assigning the IP)
2. **As Authorized Representative of Fox ML Infrastructure LLC** - Assignee (the company receiving the IP)

### Step 2: Create PDF and Add Digital Signatures

**Digital Signature Method (Recommended - Free and Legally Valid)**

**Free Options:**

**Option A: Using Adobe Acrobat Reader (Free)**
1. Convert the markdown file to PDF (or copy text into Word/Google Docs and export as PDF)
2. Open the PDF in Adobe Acrobat Reader (free download)
3. Go to Tools → Fill & Sign
4. Click "Sign" → "Add Signature"
5. Choose "Draw" to draw your signature with mouse/trackpad, or "Type" to type your name
6. Place signature in both signature fields
7. Save the PDF

**Option B: Using Online PDF Signers (Free)**
1. Convert to PDF
2. Use a free online service like:
   - SmallPDF.com (free PDF signer)
   - PDF24.org (free PDF tools)
   - ILovePDF.com (free PDF signer)
3. Upload PDF, add signatures, download signed PDF

**Option C: Using Preview (Mac) or Built-in PDF Tools**
- **Mac**: Open PDF in Preview → Tools → Annotate → Signature → Create Signature
- **Windows**: Use built-in PDF viewer or Microsoft Edge (has annotation tools)

**Option D: Simple Method - Type Your Name**
1. Convert to PDF
2. Open in any PDF editor (even free online ones)
3. Use "Add Text" tool to type your name in signature fields
4. This is legally valid as an electronic signature
5. Save the PDF

**All of these methods create legally valid electronic signatures per Section 9 of the agreement.**

### Step 5: Store Securely

**Store signed copies in:**
1. **Physical copy** (if printed) - In a secure file or safe
2. **Digital copy** - In a secure location (encrypted drive, secure cloud storage)
3. **Business records** - Keep with your LLC formation documents

**Recommended locations:**
- Business file cabinet (physical)
- Encrypted cloud storage (Google Drive, Dropbox, etc.)
- Business records folder on your computer
- Consider keeping a copy with your accountant or lawyer

## Important Notes

### Why This Matters

This agreement ensures that:
- **All IP you've created** is owned by Fox ML Infrastructure LLC (not you personally)
- **The LLC can license and monetize** the software without IP ownership disputes
- **Enterprise clients** see clear IP ownership (important for procurement)
- **Future investors or partners** see clean IP structure

### Legal Validity

- **Electronic signatures are valid** (per Section 9 of the agreement)
- **You can sign in both capacities** (you're the owner/authorized representative)
- **No witnesses required** (though you can have one if you want extra formality)
- **Notarization not required** (digital signatures are legally valid without notary)
- **Typed signatures are valid** (you can even just type your name - no need to draw)

**See `LEGAL/IP_ASSIGNMENT_QUICK_SIGN.md` for fastest free method using online PDF signers.**

### If You Want Extra Formality

You can:
- Have a witness sign (optional)
- Get it notarized (optional, but adds extra legal weight)
- Have a lawyer review it (recommended if you have concerns)
